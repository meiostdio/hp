JDKはいろいろある

Javaを開発するうえでJDK(Java Development Kit)をインストールしなければなりません。
JDKはソースコードのコンパイラ、Java仮想マシンなどが含まれます。
JDKは色んな団体が公開していて、お金を払わなければインストールできないものから無料で使えるものがあります。
私が使っているものはOracleJDKというOracle社が公開している無償のJDKです。(サーバーとして不特定多数が使用する場合は有償ライセンスが必要らしい)
わざわざインストールしなおすのが面倒なのでやっていませんが、私がオススメするのはOpenJDKです。OpenJDKはオープンソースでありすべての機能において無料で使用することができるからです。

そもそも提案(JEP)とは？

そんなOpenJDKは現在のバージョンが20
現在バージョン21を開発中ですが、OpenJDKプロジェクトに参加している開発者は誰でも新機能について提案できます。(JEP-Java Enhancment Proposalといいます)
この提案は他のメンバーによって精査されて採択するか決められるのですが、その中に面白いと感じた提案がありました！
私もGoogle翻訳で読んでいて大まかに意味が分かるのでぜひ読んでみてください。
OpenJDK JEP 445: JDK 19

提案の内容

ここからは提案の内容に触れていきます。 まず以下のコードはJavaのHelloWorldです。

public class HelloWorld { 
    public static void main(String[] args) { 
    System.out.println("Hello, World!");
    }
}

提案者はこのコードに対してめっちゃ文句を言っています。
提案の文をGoogle翻訳した文↓

ここには、プログラムの動作に対して、コードが多すぎ、概念が多すぎ、構成要素が多すぎて、混乱が多すぎます。 Google翻訳
確かに初めて勉強をしたときに理解できない部分がとても多いですね。
